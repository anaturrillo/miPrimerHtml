Desafío!

  • Crear los diferentes documentos HTML que conforman las diferentes secciones del sitio web del proyecto final.
  • Agregar una lista con el Menú Principal del sitio web (nombre de las diferentes secciones y el link pertinente a la pagina que corresponda).
  • Agregar una tabla para mostrar información pertinente. Agregar la imagen de la tapa de cada disco o item que corresponda según el tema elegido.
  • Agregar un formulario de contacto en la sección que corresponda. El formulario debe contar con al menos un menú desplegable, un checkbox, un botón para resetear la información introducida y un campo de e-mail.
